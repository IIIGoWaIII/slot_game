# Slot Game - Custom Artwork Showcase

A slot machine game built with **PIXI.js** to demonstrate custom sprite work and artwork capabilities for slot games.

## About This Project

This project is a fork of [MYSTICAL FOREST ADVENTURE](https://github.com/Akash-Gher/MYSTICAL-FOREST-ADVENTURE) by Akash Gher.

**Purpose**: Showcase custom sprite artwork to demonstrate capability in creating artwork for slot games. This serves as proof of ability to create professional slot game graphics.

## Features

- Preloader with progress bar and animated logo
- 3x5 slot reels with customizable symbols
- Bet controls with increase/decrease buttons
- Spin functionality with balance and win tracking
- Game information popup (paytable) using a modal
- Responsive design for different screen sizes

## Tech Stack

- **PIXI.js** - Rendering engine
- **GSAP** - Animations
- **Howler.js** - Sound effects

## Running Locally

### Option 1: Python Simple Server
```bash
python -m http.server 8080
```
Then open `http://localhost:8080`

### Option 2: VS Code Live Server
Install the Live Server extension and click "Go Live"

## Deploying

This project is configured for automatic deployment to **GitHub Pages** on push to master.

1. Push to GitHub
2. Go to Repository Settings > Pages
3. Select "Deploy from a branch"
4. Choose "gh-pages" branch
